Technologies

  • AngularJS

    AngularJS is a free and open-source JavaScript-based web framework for developing single-page applications.

  • Django

    Django is a free and open-source, Python-based web framework that runs on a web server. It follows the model-template-views (MTV) architectural pattern.

  • Bootstrap

    Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains HTML, CSS, and (optionally) JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

  • ReactJS

    ReactJS is a free and open-source front-end JavaScript library that aims to make building user interfaces based on components more seamless.
